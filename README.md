# Virtual-Routing

## 环境说明
* 开发环境：Linux、Windows 10
* 运行环境：Linux、Windows 10
* 运行方式：python router.py
* 使用库：socket、os、threading、operator、time

## 实现逻辑
* 去中心化路由：LS算法、DV算法
* 中心化路由：LS算法

## 源码
https://github.com/RealEasyin/Virtual-Routing

## 功能

### 自动功能
* 自动更新路由表

### 交互功能
* 添加新邻居 
* 模拟路由器脱离网络
* 显示路由表
* 显示从源到目的地的路由路径
